## Description

Flower Power is an interactive visual simulation created using Python and the Pygame library. The program allows users to 
click on the grass area of the screen to plant flowers that grow dynamically over time. Each flower is procedurally generated 
with randomized colors, petal counts, and growth heights, making every garden unique.

The project uses multiple functions to organize drawing, flower creation, growth updates, event handling, and animation. Flowers gently sway back and forth using sine wave motion to create a more natural and lively appearance. Additional environmental details such as clouds, a sun, title text, grass, and a sky background help create a more polished visual scene.

This project demonstrates several programming concepts including:
- functions
- loops
- conditionals
- dictionaries
- lists
- animation
- randomization
- user interaction with mouse input
- mathematical calculations using the math module

Users interact with the program by clicking anywhere on the grass area of the screen. After clicking, flowers appear and continue growing until they reach their maximum growth value. Because the flower properties are randomized, no two flowers look exactly the same.
